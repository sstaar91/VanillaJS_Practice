# RESAT Farming Challenge

- [미니인턴](https://miniintern.com/)에서 진행한 챌린지 결과물 레포지토리입니다.
- 프론트엔드 개발자라면 구현해봐야 하는 UI로 구성된 미션을 10일 동안 해결하는 챌린지입니다.

## 챌린지 관련 기본 정보

- 챌린지 기간 : 2023.06.19 ~ 30
- 사용 스택 : html5 / css3 / javascript
